#### ndarray重要属性：
* ndim: 维度，轴的数量
* shape: 形状
* size: 元素数量
* dtype: 元素类型
* itemsize: 每个元素大小(bytes)
* data: 数据存储区

#### Copies & Views
###### 1. No Copy at All
简单的赋值操作不会产生任何Copy
~~~ Python
>>> a = np.arange(12)
>>> b = a
>>> b is a
True
~~~
###### 2. View or Shadow Copy
view方法会产生一个新的array对象，但是会共享data数据
~~~ Python
>>> b = a.view()
>>> b is a
False
>>> b.base is a
True
~~~
###### 3. Deep Copy
copy方法会产生一个array对象，data数据也会新复制一份
~~~ Python
>>> b = a.copy()
>>> b is a
False
>>> b.base is a
False
~~~
